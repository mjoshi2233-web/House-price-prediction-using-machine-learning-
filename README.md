# House-price-prediction-using-machine-learning-

A Machine Learning project that predicts California house prices using the XGBoost Regressor algorithm.
This project was built as part of my Machine Learning learning journey and helped me understand how regression algorithms can be applied to predict continuous values from real-world housing data.
Concepts Used

->Data Loading and Exploration
->Exploratory Data Analysis (EDA)
->Correlation Analysis and Heatmaps
->Feature and Target Selection
->Train-Test Split
->XGBoost Regressor
->Model Evaluation (R² Score, MAE)
->Visualization of Actual vs Predicted Values

Results:-

Trained an XGBoost Regressor on the California Housing dataset.
Achieved strong predictive performance, evaluated using R² score and Mean Absolute Error.
Visualized actual vs predicted house prices to assess model accuracy on both training and test data.
